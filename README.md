# Портфолио сайт

Профессиональный сайт-портфолио, созданный с использованием современных веб-технологий.

## Технологический стек

- **TypeScript** - для типобезопасности кода
- **React** - для создания пользовательского интерфейса
- **Vite** - для быстрой сборки и разработки
- **Express** - для backend API
- **Multer** - для загрузки файлов
- **Jest** - для модульного тестирования
- **React Testing Library** - для тестирования React компонентов

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

**Вариант 1: Запустить frontend и backend вместе**
```bash
npm run dev:all
```

**Вариант 2: Запустить отдельно**
```bash
# Terminal 1 - Frontend (Vite)
npm run dev

# Terminal 2 - Backend (Express)
npm run dev:server
```

- Frontend будет доступен по адресу `http://localhost:3000`
- Backend API будет доступен по адресу `http://localhost:3001`
- Админка доступна по адресу `http://localhost:3000/admin`

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist`

### Запуск в продакшене

```bash
npm start
```

Сервер будет запущен на порту 3001 (или PORT из переменных окружения) и будет обслуживать как API, так и статические файлы из `dist`.

## Структура проекта

```
src/
├── components/          # React компоненты
│   ├── Header/         # Шапка сайта
│   ├── Hero/           # Главная секция
│   ├── Projects/       # Проекты/Галерея
│   ├── Contact/        # Контакты
│   └── Footer/         # Подвал
├── admin/              # Админ-панель
├── lib/
│   ├── api.ts          # API клиент для работы с сервером
│   └── storage.ts      # Утилиты для localStorage (legacy)
├── data/               # Дефолтные данные
└── App.tsx             # Главный компонент

server/
├── index.js            # Express сервер
└── data/               # Данные и изображения (создаётся автоматически)
    ├── projects.json   # JSON файл с проектами
    └── images/         # Папка с загруженными изображениями
```

## API Endpoints

- `GET /api/projects` - получить все проекты
- `POST /api/projects` - сохранить проекты
- `POST /api/upload` - загрузить изображение
- `DELETE /api/images/:filename` - удалить изображение
- `GET /api/health` - проверка работоспособности сервера

## Переменные окружения

Создайте файл `.env` для настройки:

```env
VITE_API_URL=http://localhost:3001
PORT=3001
```

## Тестирование

### Запуск всех тестов

```bash
npm test
```

### Запуск тестов в режиме наблюдения

```bash
npm run test:watch
```

### Запуск тестов с покрытием

```bash
npm run test:coverage
```

## Деплой на хостинг

1. Соберите проект: `npm run build`
2. Убедитесь, что папка `server/data` существует и доступна для записи
3. Запустите сервер: `npm start`
4. Настройте переменные окружения (PORT, и т.д.)
